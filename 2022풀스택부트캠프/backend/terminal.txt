터미널 명령어

ls : (list) 현재 디렉토리 파일 리스트를 보여준다.
ls -l : 파일 리스트 정보를 상세히 보여준다.
ls -a : 숨겨진 파일 리스트를 포함해서 보여준다.
ls -al : 숨겨진 파일 리스트를 포함해서 상세한 정보를 보여준다.

pwd : 현재 경로 위치를 보여준다.

cd + (폴더이름) : 디렉토리를 변경할때 사용. 지정한 폴더로 이동한다. - 첫글자를 쓰고, Tab을 누르면 자동으로 입력된다.
cd .. : 한단계 이전 폴더로 이동한다.
cd / : 가장 처음 단계 폴더로 이동한다. 루트 디렉토리
cd ~ : 홈 디렉토리로 이동.

clear : 터미널에 보이는 내용을 모두 지운다.

mkdir (폴더이름) : 디렉토리 폴더를 만든다. 띄어쓰기를 이용해서 여러개를 추가할 수 있다.
mkdir -v (폴더이름) : 디렉터리 폴더를 만들고 내용을 출력한다.

touch (파일이름) : 확장자를 포함한 파일을 만든다. 띄워쓰기를 하면 여러 파일을 만들 수 있다. 확장자 이름을 명확하게 적어야 한다.

rm : 파일을 삭제한다. (휴지통으로 이동이 아니라 영구삭제한다.)
rmdir : 비어있는 폴더를 삭제한다.
rm -rf (폴더이름): 폴더 전체를 삭제한다.

q : 현재 실행된 명령어를 종료한다.
